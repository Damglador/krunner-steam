# krunner-steam

Unofficial Steam plugin for KRunner.

Actions for search results:
- Steam Library - opens the game in your Library in Steam
- Community Hub
- Local Files

## Installation

### AUR

```sh
yay -S krunner-steam-git
```

### Manual

```sh
cd "${XDG_DATA_HOME:-$HOME/.local/share}/"
git clone https://github.com/Damglador/krunner-steam
cd krunner-steam
./install.sh
```
